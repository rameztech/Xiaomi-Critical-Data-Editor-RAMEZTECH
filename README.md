🚀 أبرز المميزات والتقنيات:
⚙️ دعم كامل للمعالجات:
تعمل الأداة بكفاءة على كلا النوعين الأكثر شيوعاً: MediaTek (MTK) و Qualcomm (QCOM) مع خوارزميات تشفير مخصصة لكل منهما.

📄 استخراج وتعديل فوري:
قم بسحب ملف critical_data.bin واستخراج جميع الحقول الحيوية (IMEI1، IMEI2، CPU ID، Product ID، WiFi MAC، Bluetooth MAC) وتعديلها بنقرة زر.

📤 رفع ذكي للطلبات إلى السيرفر (Telegram Bot):
إرسال الطلبات مباشرة إلى خادم المعالجة المركزي الخاص بنا (عبر بوت تيليجرام)، مع تحديث فوري لحالة الطلب.

📂 متابعة الملفات عبر "ملفاتي":
لوحة تحكم متكاملة تعرض جميع طلباتك السابقة، مع حالتها (قيد التقدم / جاهز)، وإمكانية تحميل الملفات الجاهزة مباشرة عبر المتصفح.

🔄 تحديث تلقائي عبر الإنترنت (Auto-Update):
مجهزة بنظام تحديث أوتوماتيكي عن طريق GitHub Releases، ليصلك أحدث الإصدارات والتحسينات دون عناء.
